# **App Name**: MarineLogix

## Core Features:

- Offline Engine Log Recording: Record engine activity for up to 4 hours with data input similar to provided Excel examples, stored locally for offline access.
- Automatic Data Synchronization: Automatically synchronize locally stored data across all 5 users when an internet connection is available, ensuring data consistency and updates.
- Simplified Authentication: Implement Better Auth for one-time login. Session data is saved locally for persistent access.
- Inventory Management: Manage stock levels, track usage, and automatically update inventory across all synchronized users.
- Centralized Activity Log: Aggregate and organize engine activity logs from all users into a single, synchronized view for efficient monitoring and analysis.
- Inventory Usage Tracking: Track inventory usage and automatically update stock levels across all synchronized users.
- Customizable Settings: Configure ship name, officer names, and other key settings to reflect the specific operational context.

## Style Guidelines:

- Primary color: Deep blue (#1A237E) to convey reliability and maritime tradition.
- Background color: Light blue-gray (#D2D6EB) for a calm, readable interface.
- Accent color: Vibrant yellow (#FFD700) for key actions and alerts to draw attention effectively.
- Body and headline font: 'Inter', a grotesque-style sans-serif for a modern and neutral feel.
- Use clear, technical icons to represent engine components, inventory items, and user actions.
- Prioritize key metrics on the homepage dashboard for quick status assessment. Use a clean, tab-based interface.
- Subtle transitions to indicate data synchronization and updates to improve the user experience.
